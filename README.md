# System – Telegram Importer

**Joomla! plugin for automatic import of Telegram channel posts as articles.**

## Features
- Imports Telegram channel posts into Joomla articles (com_content)
- Supports images, categories, custom author, import interval and post history deduplication
- Multilanguage (EN, UA), tested on Joomla! 5.2+

## Installation & Configuration

1. Download ZIP-pack or use sources.
2. Go to Joomla Admin → Extensions → Install → Upload your package file.
3. Enable plugin in Extensions → Plugins.
4. Configure:
   - **Bot Token:** from @BotFather  
   - **Channel ID:** (format: -100xxxxxxxxxx)
   - **Category, Author**: for imported articles
   - **Other settings:** see plugin options
