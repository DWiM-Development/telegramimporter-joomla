# Telegram Importer

**Joomla! plugin for automatic import of Telegram channel posts as articles.**

## Features
- Imports Telegram channel posts into Joomla articles (com_content)
- Supports images, categories, custom author, import interval and post history deduplication
- Multilanguage (EN, UA), tested on Joomla! 5.2+

## Installation & Configuration

1. Download ZIP-pack or use sources.
2. Go to Joomla Admin → Extensions → Install → Upload your package file.
3. Enable plugin in Extensions → Plugins.
4. Configure:
   - **Bot Token:** from @BotFather  
   - **Channel ID:** (format: -100xxxxxxxxxx)
   - **Category, Author**: for imported articles
   - **Other settings:** see plugin options


# Telegram Importer
**Плагін Joomla! для автоматичного імпорту постів із Telegram-каналу в матеріали.**

## Можливості
Імпортує пости Telegram-каналу у матеріали Joomla (com_content)
Підтримує зображення, категорії, власного автора, інтервал імпорту та унікальність постів
Багатомовний (UA, EN), протестовано на Joomla! 5.2+

## Встановлення та налаштування
1. Завантажте ZIP-архів або використайте вихідний код.
2. В адмін-панелі Joomla перейдіть: Розширення → Встановити → Завантажити файл пакету.
3. Активуйте плагін: Розширення → Плагіни.
4. Налаштуйте:
   - **Bot Token**: отримати у @BotFather
   - **Channel ID**: (формат: -100xxxxxxxxxx)
   - **Категорія, Автор**: для імпортованих матеріалів
   - **Інші налаштування**: див. опції плагіна
